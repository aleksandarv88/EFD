0. FRAMEWORK
Base logic     : classical first-order with sets
Universe P     : minimal self-consistent closure of difference
Difference Δ   : forced by primal loop (nothing ↔ possibility)

    **full superposition of all possibilities**.

1. PRIMITIVES
------------------------------------------------------
(i) primal mutual-definition loop:
        (nothing ↔ possibility)
(ii) existence of a nontrivial difference operator Δ

From (i):
    “absence” and “presence” arise together by mutual definition.

From (ii):
    Δ(a,b) ⇔ a ≠ b

2. DERIVED NON-EMPTINESS
------------------------------------------------------
(G2) Non-empty (derived):

4. BASE DIFFERENCE
------------------------------------------------------
Bind logic to math via:
    a := nothing
    b := possibility

Base relation:
    Δ¹ := { (a,b) }

Higher relations:
    Δⁿ⁺¹ := { (u,v) | u,v ∈ Δⁿ, u ≠ v }

Interpretation update:

Ω is static:

7. BOUNDARY AND NOISE
------------------------------------------------------

Define:
    ∂M := (Ω \ Coherent(M)) ∩ N(M)

where N(M) is Δ-neighborhood.

Noise η measures non-reconciliation at ∂M.
